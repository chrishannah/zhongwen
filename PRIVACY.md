# Privacy Policy

**Zhongwen** is a Chrome extension that displays Mandarin Chinese characters on your new tab page.

## Data Collection

Zhongwen does **not** collect, store, or transmit any personal data.

## Local Storage

The extension stores your preferences (selected word categories and Unsplash API key) locally on your device using Chrome's storage API. This data never leaves your browser.

## Third-Party Services

If you choose to enable background images by entering your own Unsplash API key, the extension will make requests to the [Unsplash API](https://unsplash.com/api) to fetch landscape photographs. This is entirely optional and requires you to opt in by providing your own key. No data is sent to Unsplash beyond the image request itself.

## Permissions

- **Storage**: Used to save your word category preferences and cached background image data locally.
- **Unsplash API access** (optional): Only requested if you enter an Unsplash API key. Used solely to fetch background images.

## Contact

If you have questions about this privacy policy, please open an issue on the [GitHub repository](https://github.com/chrishannah/zhongwen).
